Project React


Dit is een project dat wij moesten maken voor het vak React. In dit project mochten wij over een zelfgekozen thema een webpagina bouwen. Hierin heb ik 3 verschillende pagina's gemaakt: een home page, een randomizer en een search page. De homepage is een simpele navigatie pagina waarin de gebruiker de keuze heeft naar welke pagina hij wilt. Op de randomizer krijgt de gerbuiker een random kaart te zien en op de search page kan je een "archetype" opzoeken en krijg je elke kaart hiervan te zien. Dit is wel degelijk een project waarop ik trots ben omdat dit een vak is waarin ik niet goed ben.

Project PHP


Dit is een project dat wij moesten maken voor het vak PHP. Deze site is gebouwd in PHP en maakt gebruik van tailwind. In dit project kregen wij een customer case die wij moesten uitwerken. Wij moesten als customer case een webapplicatie bouwen voor WTC Roland, deze geven jaarlijks een koers. Deze werd dan gebruikt door de mensen van de vereneging en een deel door de algemene gebruiker. Hetgene dat ik heb gedaan in dit groepsproject is:
  • De eerste versie van header & footer
  • Info opzoeken naar een mailserver
  • Locale mailserver aanmaken en 1ste test
  • Mails beheren pagina
  • Inschrijvingstermijn pagina
  • Volledige versie van klassement pagina
  • Liveklassement pagina
  • Tijdsregistratie pagina
  • Indelen van motoren pagina
  • Startvolgorde pagina
  • Inschrijvingsgeld pagina
